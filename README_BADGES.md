## How to Use

1. Copy the badges you want
2. Replace `YOUR-USERNAME` and `REPO-NAME`
3. Add to the top of your README.md (after the title)
4. Commit and push:
   ```bash
   git add README.md
   git commit -m "Add badges to README"
   git push
   ```
