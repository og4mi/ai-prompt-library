# Reload Extension

To see the updated styling changes:

1. Go to `chrome://extensions/`
2. Find "Prompt Library" extension
3. Click the refresh/reload icon (circular arrow)
4. Click the extension icon in your toolbar to see the new design

The extension now matches the main app's design system with:
- Same color scheme (HSL color variables)
- Same primary blue color
- Consistent border styles and hover effects
- Matching typography
- Similar card designs and spacing
