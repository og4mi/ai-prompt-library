# Quick Start Guide

Get up and running with AI Prompt Library Manager in 3 steps:

## 1. Install Dependencies

```bash
npm install
```

## 2. Start Development Server

```bash
npm run dev
```

## 3. Open in Browser

Navigate to [http://localhost:3000](http://localhost:3000)

---

## First Steps

Once the application loads:

1. **Add Your First Prompt**
   - Click the "Add Prompt" button in the top right
   - Fill in the title and prompt content
   - Select a category and AI model
   - Add tags for better organization
   - Click "Add Prompt"

2. **Try the Search**
   - Use the search bar to find prompts
   - Search works across titles, content, tags, and notes
   - Fuzzy matching helps find prompts even with typos

3. **Filter Your Prompts**
   - Use the sidebar to filter by:
     - Categories
     - AI Models
     - Tags
     - Favorites only

4. **Customize Your View**
   - Toggle between grid and list view
   - Switch between light and dark themes
   - Sort prompts by date, alphabetical, usage, or favorites

## Production Build

```bash
npm run build
npm start
```
